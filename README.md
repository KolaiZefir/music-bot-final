# Music Player Telegram Mini App

## 📋 Описание
Музыкальный плеер для Telegram Mini App с бэкендом на Flask и фронтендом на HTML/CSS/JS.

## 🚀 Быстрый старт

### Бэкенд (Render)
1. Создайте Web Service на Render
2. Подключите GitHub репозиторий
3. Укажите:
   - Root Directory: `backend`
   - Build Command: `pip install -r requirements.txt`
   - Start Command: `gunicorn web_app:app`
4. Добавьте переменные окружения (если нужно)

### Фронтенд (Vercel)
1. Создайте проект на Vercel
2. Подключите папку `music-frontend`
3. Дождитесь деплоя
